# Face Search  

## Requirements  
- Python 3.x  
- Flask  
- OpenCV  
- face_recognition  

## Setup  
1. Install dependencies:  
    ```bash  
    pip install flask opencv-python face_recognition  
    ```  

2. Start the server:  
    ```bash  
    python app.py  
    ```  

3. Access the website:  
    - Open `http://<your-ip>:5000` on any device in the same network.  

## Features  
- Upload selfies directly from the web camera.  
- Search for similar faces in a predefined folder.
